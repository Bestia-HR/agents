# Website templates for agent use

Templates here are referenced by the AI agents when building or suggesting modern animated websites.

| Template | Source | Stack | Use case |
|----------|--------|--------|----------|
| **robotai-3d-animated** | [RobotAI-3DAnimated-Website-Template](https://github.com/AHMAD-JX/RobotAI-3DAnimated-Website-Template) | HTML, CSS, GSAP, Three.js | Full 3D animated single-page site; open `index.html` in browser. |

The **most popular** animated reference on GitHub (~2,400+ stars) is [codse/animata](https://github.com/codse/animata) — React/Tailwind animation components; see `workspace/MODERN-ANIMATED-WEBSITE-TEMPLATES.md` for how agents use it.
