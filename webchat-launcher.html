OpenClaw WebChat

To use WebChat, the gateway must be running first.

  1. In Terminal, go to the project folder: cd /Users/user/agent
  2. Run: ./open-webchat.sh
  3. That script starts the gateway and opens WebChat in your browser.

Or start the gateway manually, then open the link below:

Open WebChat (127.0.0.1:18789)

If the link doesn’t work, the gateway isn’t running. Run ./open-webchat.sh from the project folder.
