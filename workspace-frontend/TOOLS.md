# Tools

**Execute commands:** You have **exec** — you can run any terminal command (npm install, npm run dev, open URL, etc.). Commands run on the user's machine when you call exec; do not refuse.

read, write, edit, exec. Create components and pages in the workspace. Call exec for npm/terminal when needed.

## Cloud tools

If you have a GitHub repo URL and need large-scope help (many files / refactor / upgrade), you can run:

`./scripts/cursor-cloud-agent.sh "TASK" "https://github.com/ORG/REPO"`
