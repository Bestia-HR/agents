# Learning references (project overview)

This project’s AI agents use the references below when users ask for learning resources or when citing authoritative sources. The same list is available inside each agent workspace as `LEARNING-REFERENCES.md`.

## Quick list

| Area | Main resources |
|------|----------------|
| **Web (HTML, CSS, JS)** | [MDN Learn](https://developer.mozilla.org/en-US/docs/Learn_web_development), [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web), [freeCodeCamp](https://www.freecodecamp.org/learn), [web.dev](https://web.dev) |
| **Animation & 3D** | [GSAP](https://gsap.com/docs/v3/), [Three.js](https://threejs.org/docs/), [web.dev animations](https://web.dev/articles/animations-guide) |
| **React** | [React docs](https://react.dev), [awesome-react-components](https://github.com/brillout/awesome-react-components), [Vite](https://vitejs.dev) |
| **Accessibility** | [W3C WAI](https://www.w3.org/WAI/), [MDN Accessibility](https://developer.mozilla.org/en-US/docs/Web/Accessibility), [web.dev a11y](https://web.dev/accessibility/) |
| **Backend / APIs** | [MDN HTTP](https://developer.mozilla.org/en-US/docs/Web/HTTP), [freeCodeCamp Backend](https://www.freecodecamp.org/learn/back-end-development-and-apis/) |
| **Layout** | [MDN CSS layout](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout), [CSS-Tricks guides](https://css-tricks.com/guides/) |
